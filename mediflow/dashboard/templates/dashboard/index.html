🎯 About This Project

MediFlow is a real-time hospital operations intelligence platform built to demonstrate full-stack development capabilities with AI integration. Inspired by SOPEONOW — a Singapore-based healthcare intelligence company — this project mirrors their core product vision: giving hospital administrators a 360° live view of their entire hospital.

The platform monitors patient flow, bed availability, department queues, and staff operations in real time — and uses Groq's LLaMA AI model to answer natural language questions about live hospital data, implementing a RAG (Retrieval Augmented Generation) architecture pattern.

⚡ Key Features
📊 Live Dashboard
Real-time patient metrics, bed status, department queues with auto-refresh every 30 seconds
🚨 Auto-Alert System
Python automatically detects patients waiting over 30 minutes and fires critical alerts
🤖 AI Chatbot
Groq LLaMA AI reads live hospital data and answers natural language questions in context
🔐 Role-based Login
Secure authentication system with Django's built-in auth framework
📈 Charts & Analytics
Zone distribution, bed utilization, and department queue charts powered by Chart.js
🌙 Dark / Light Theme
Professional theme toggle with localStorage persistence across sessions

"I studied SOPEONOW's product and built MediFlow — a miniature version of their core concept. It has a real-time operations dashboard, an auto-alert system that fires when patients wait too long, department queue monitoring, and an AI assistant powered by Groq's LLaMA model that reads live hospital data before answering — implementing RAG architecture.
